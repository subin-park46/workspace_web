window객체

	프로퍼티
	 -document : 웹 페이지 그 자체
	 -history : 브라우저의 사용자 히스토리를 목록, 배열 상태로 저장하는 객체
	 -location : url 정보를 가져오는 객체 (location.href = 새로운 페이지로 이동)
	 -navigator : 현재 브라우저의 정보를 반환하는 객체
	 -screen : 사용자 화면에 대한 정보
	 -frames : 프레임들은 각각의 window 객체를 별도로 가짐
	 -parent : 해당 요소의 바로 위에 존재하는 하나의 부모 요소 반환
	 -top : 최상위 창 반환
	 -self : 현재 자신의 창 반환
	메서드
	 -alert() : 경고창 띄우는 메서드
	 -confirm() : 확인 / 취소 버튼 선택 가능한 대화상자
	 -prompt() : 입력창 / 확인 / 취소 대화상자
	 -back() : 한 단계 전 화면으로 돌아감
	 -forward() : 한 단계 다음 화면으로 넘어감
	 -setInterval() : 일정한 시간 간격을 두고 반복해서 실행하고 싶을 때
	 -clearInterval() : setInterval의 반복실행을 중단시키는 메서드
	 -setTimeout() : 일정시간이 지난 후 실행 해야 하는 경우 사용
	 -open() : 새로운 창 반환, 실패 시 null
	 -close() : 현재 창 닫는 메서드, 닫을 창 지정 가능
	 -scroll() : 스크롤 이동, 스크롤 실행 시 이벤트 발생
	 -scrollBy() : 현재 위치에서 부여된 값만큼 스크롤 이동
	 -scrollTo() : 해당 좌표로 스크롤을 이동시킴
